# Analyzing Crime in Los Angeles

## About
An exploratory data analysis of crime incidents in Los Angeles. The project explores patterns in crime timing, location, and victim demographics.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## What I Did
- Identified which hour of the day has the highest frequency of crimes
- Found the area with the largest volume of night crimes (10 PM – 3:59 AM)
- Analyzed the distribution of crimes across different victim age groups

## Dataset
Crime data provided by DataCamp, originally sourced from the Los Angeles Police Department (LAPD) public records.
